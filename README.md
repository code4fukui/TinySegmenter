# TinySegmenter
 
## demo

https://code4fukui.github.io/TinySegmenter/

## how to use

```js
import { TinySegmenter } from "https://code4fukui.github.io/TinySegmenter/TinySegmenter.js";

const segs = TinySegmenter.segment("私の名前は中野です");
console.log(segs);
```
